## Feature Categories

### Core E-Commerce
- Product management and catalog
- Shopping cart and checkout
- Order processing and tracking
- Payment integration (Stripe + COD)

### User Management
- Customer accounts via Clerk
- Address management
- Order history
- Wishlist functionality

### Employee Portal
- Role-based access control
- Order processing workflow
- Cash collection system
- Performance analytics

### Administration
- Product CRUD via Sanity Studio
- Order management
- Customer management
- Analytics dashboard
